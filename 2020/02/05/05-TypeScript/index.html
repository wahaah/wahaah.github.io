TypeScript

介绍

typescript = type + ES6

ts 是 js 的强类型版本，然后在编译期去掉类型和特有语法，生成纯粹的js代码，最终浏览器运行的仍然是js，所以ts并不依赖于浏览器的支持，不会带来兼容性问题。

ts支持所有的js 语法，是js的超集

TS是一种类似于js的语言，在js的基础上增加了类型，同时增加了js部分语法功能，遵循es6标准规范

Angular2 采用ts编写

搭建ts环境

用angular脚手架搭建的项目不用搭建typescript环境，因为他已经集成在里面了，单独学习需要搭建环境。

环境的两种方式：

1.在线编译（ts官网中）

2.本地搭建环境

TS数据类型

建议多使用let(变量)和const(常量)，不建议使用var

ES6常量一旦定义不允许被重新赋值

  • 布尔值 boolean

    let isNaN: boolean = true

  • 数值型 number

    let num: number = 15

  • 字符串 string

    let str: string =“你好，我是哥哥”

  • 数组 array

  • 元组

元组用于表示一个已知元素数量和类型的数组，各元素的类型不必相同

  • 对象 Object

允许赋任意值

但不能调用任意方法，即便他真的有

  • any

任意类型，不确定类型的情况下使用，但尽量不要使用

  • void

类型除了用于变量，还可用于函数的形参

当函数没有返回值的时候，用void，代表没有返回值

  • null
  • undefined
